Styling Specyfic Objects

  • Wy have focused on type selectors.
  • What if you don't want to style all of the links, just some? or just some of the lists?
  • CSS gives options

CSS selectors

  • Some selectors follow the DOM
  • Descendant selectors (nav a)
    • Style all of the anchor links inside a nav tag
  • Child selectors (nav > a)
    • More constraining, the anchor elements must be a child of the nav, no intermediate tags, e.g. paragraph
  • Adjecent sibling (hl + ol)
    • Elements must be at same level and follow each other

# id selectors

  • Used to identify a single element in the DOM, shoud be unique.
  • Was used extensively for <div id = "header">, <div id="footer">, etc
  • There is a small movement to move the use of id out of CSS. Id is used for javascript too.

Class selector

  • .class selector
    • Used to identify an element in the DOM that is part of a special class items.
    • Think of thumbnail images, all of the links that are in the navigation, your social media images etc...

Narrowing the scope

  • As you get more advanced pages, you will want to narrow the scope of the action.
  • p.main - paragraph using the "main" class
  • header img.special - paragraph inside header that use special class

Expanding the scope

  • You can combine elements with a comma
    • p, hl, #mail, .special{... rules to apply to all of them ...}
  • Review: What happenes when there are multiple rules for the same selector?
    • When there are conflicts, use the one processed most recently
    • UNLESS a rule has !important

More Attribute Selectors

  • Universal
    • * applies to every element on the page.
    • Ackk!! Try this!
  • Attribute Selectors
    • a[href='info.html'] - styling every link that contains info.html.
  • Pseudo-Classes
  • Pseudo-Elements

Attribute selectors

  • You may want to search the DOM for certain elements that have an attribute you are looking for.
    • All of the images that use gif files...
    • All of images that have empty alt text...
    • All of the links that go to government sites...

Using operators

  • Operators can be used to find those attribute values you are looking for.
    • ^: match the beginning exactly a[href^='http://umich']
    • $: match the end exacly img[src$='png'] apply to .png images.
    • *: wildcard a [href*='umich']

The Good News

You can use style sheets from others to style your code, just by adding classes

You can override style sheets from others just by rewriting the class, or making your own version of it and linking it lasy.
